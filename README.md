# SK Education - Study Materials Platform

A modern educational platform offering study materials across multiple subjects including Science, Mathematics, Hindi, Sanskrit, and Social Studies.

## Features
- 📚 Multiple categories and subcategories
- 📄 PDF study materials
- 🎥 Video lectures
- 👤 User authentication
- 📊 Admin panel for content management
- 📱 Fully responsive design
- 🔔 Telegram notifications

## Tech Stack
- HTML5
- CSS3
- JavaScript
- Firebase Realtime Database
- Telegram Bot API

## Deployment

### GitHub Pages
1. Push all files to a GitHub repository
2. Go to repository Settings > Pages
3. Select main branch as source
4. Your site will be live at `https://[username].github.io/[repo-name]`

### Vercel
1. Install Vercel CLI: `npm i -g vercel`
2. Run `vercel` in project directory
3. Follow the prompts to deploy

Or connect your GitHub repository directly on [vercel.com](https://vercel.com)
